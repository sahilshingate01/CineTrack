# CineTrack 🎬

CineTrack is a modern movie watchlist web application that allows users to search for movies, save them to a personal watchlist, and manage what they want to watch next.

This project uses public movie APIs to fetch real-time movie data and provides a clean dark-themed UI inspired by streaming platforms like Netflix, Disney+, and HBO Max.

---

## 🚀 Features

* 🔍 Live Movie Search using OMDb API
* ⭐ Add / Remove movies from Watchlist
* 📌 Sticky sidebar for saved movies
* 🎲 Random Movie Night (optional feature)
* 🌙 Dark modern UI
* ⚡ Fast API-based search results
* ❌ Empty state UI for no results / empty list
